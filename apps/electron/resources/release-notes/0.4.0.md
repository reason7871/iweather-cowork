# v0.4.0 — LLM 连接、Codex 支持、工作区默认设置

多连接 AI 设置、Codex/OpenAI 支持、按工作区默认设置和文档升级。

---

## 🔌 LLM 连接（多提供商）

iWeather 现在支持多个 **LLM 连接**：

- 添加多个提供商（Anthropic、OpenRouter、Codex/OpenAI 等）
- 独立验证和管理每个连接
- 会话在第一条消息后锁定到连接

## 🤖 Codex / OpenAI 支持

通过 **Codex** 使用 OAuth 连接 OpenAI，并与 Anthropic 连接一起运行 Codex 驱动的会话。

## 🧭 按工作区默认连接

工作区现在可以设置自己的 **默认 LLM 连接**，覆盖该工作区新会话的全局默认值。

## 🎨 工作区特定默认主题

每个工作区可以通过 `defaults.colorTheme` 设置 **默认主题**，便于视觉区分上下文。

## 📚 文档更新

新增和更新的文档涵盖：

- 多连接 + 会话锁定
- 按工作区默认连接 + 主题
