# 行为准则

## 我们的承诺

我们致力于为每个人提供一个友好和包容的环境。

## 我们的标准

我们期望所有参与者：

- 尊重和体贴他人
- 欢迎新人并帮助他们学习
- 优雅地接受建设性反馈
- 关注对社区最有利的事情

## 执行

不可接受的行为实例可以向项目维护者报告。

所有投诉将被迅速和公平地审查和调查。

## 归属

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org/) 2.1 版本。

全文请参阅：https://www.contributor-covenant.org/version/2/1/code_of_conduct/
